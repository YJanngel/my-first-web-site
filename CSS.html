CSS이란 무엇인가?

CSS (Cascading Style Sheets)는👉 HTML 요소들의 디자인(스타일)을 정의하는 언어입니다. 즉, 색상, 글꼴, 크기, 위치, 간격, 애니메이션 등을 설정해 웹사이트를 더 보기 좋고, 사용자 친화적으로 만듭니다.
